# HappyMind – AI Mental Health Chatbot 🧠💬

HappyMind is a simple, AI-powered desktop chatbot designed to provide basic mental health support, relaxation tips, and daily encouragement. 

This application was developed as my **secondary school graduation project** with the goal of understanding the fundamentals of Machine Learning (ML) and Natural Language Processing (NLP). 

> **Note on Learning Process:** This project was built for educational purposes. The core neural network architecture and NLP pipeline were implemented and adapted using technical documentation and online tutorials. My main focus was on researching these concepts, understanding them, and successfully integrating them into a fully functional desktop application with a custom dataset.

## 🛠️ Tech Stack
* **Language:** Python
* **Machine Learning:** PyTorch (Feed-Forward Neural Network)
* **NLP:** NLTK (Tokenization, Stemming, Bag of Words)
* **GUI:** CustomTkinter (Modern desktop interface)
* **Data Format:** JSON (`intents.json`)

## ✨ Key Features
* **Custom Intent Recognition:** Uses a trained PyTorch model to classify user inputs into predefined categories (intents).
* **NLP Preprocessing:** Implements a custom pipeline to clean, tokenize, and stem user messages before feeding them to the neural network.
* **Modern GUI:** A clean, user-friendly desktop interface built with CustomTkinter.
* **Trainable Dataset:** Easy to update and retrain using the `intents.json` file and the `train.py` script.
